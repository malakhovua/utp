Отчет "Квартальный отчет по единому налогу для физ. лиц"

Отчет «Квартальный отчет по единому налогу для физ. лиц» позволяет сформировать ежеквартальный отчет субъекта малого предпринимательства - физического лица, плательщика единого налога.

Данные отчета могут быть заполнены автоматически, а также отредактированы вручную. Для ручного редактирования данных необходимо использовать кнопку "Редактировать данные ...".

Автоматически заполняются:

  • данные по перечислению единого налога, используется информация из платёжных документов с установленным признаком "Перечисление единого налога";
  • объем реализации, заполняется по данным книги доходов/расходов по единому налогу.
