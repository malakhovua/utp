Загрузка входящих налоговых документов в формате XML из "FREDO Звіт"

Обработка предназначена для пакетной загрузки входящих налоговых накладных и расчетов корректировки, полученных от контрагентов в электронном виде в формате XML через систему FREDO Звіт.

Порядок работы

  1. Таблица полученных документов в "FREDO Звіт" заполняется при открытии формы или при нажатии на кнопку "Обновить список". В таблицу попадают документы, выписанные контрагентами за указанный период по выбранной организации. Возможно также установить отбор по определенному контрагенту.
  2. После получения списка входящих документов из "FREDO Звіт" система автоматически определяет контрагента из информационной базы по коду ИНН и коду ЕГРПО(ДРФО), полученными из системы "FREDO Звіт".
  3. Если входящий документ в формате XML уже регистрировался в информационной базе (создавался входящий налоговый документ), ссылка на такой документ будет показана в таблице. Также будет произведен анализ найденного документа из информационной базы на соответствие ключевых полей: кодов контрагента, суммы НДС, номера и даты входящего налогового документа. Информация о найденных расхождениях будет показана в таблице.
  4. После загрузки списка документов из "FREDO Звіт" в таблице будут установлены флажки "загружать XML?" для тех строк для которых не были обнаружены соответствующие документы в системе продукта либо в таких документах были обнаружены расхождения в ключевых полях. Если не удалось в информационной базе найти контрагента с соответствующими данным "FREDO Звіт" кодами, флажок загрузки также будет снят.
  5. Для строк таблицы, в которых не указан контрагент необходимо его указать вручную.
  6. Загрузка XML-образов входящих налоговых документов происходит при нажатии на кнопку "Загрузить XML-Документы". Если в информационной базе нет документа регистрации входящего налогового документа, он будет создан. В таком документе будет заполнен только контрагент, номер и дата входящего налогового документа. Документы формируются текущей рабочей датой. После создания документов их можно открыть из списка. Загруженный входящий налоговый документ в XML-формате можно в дальнейшем открыть на просмотр из формы документа, или формы данной обработки, или в виде печатной формы, или в виде текстового файла XML, либо открыть документ в системе "FREDO Звіт".
  7. При установленном флажке «Сопоставление с существующими документами РВНД» доступен подбор налоговых документов с видом операции «Расчет корректировки (возврат)», введенных вручную. Подбор возможен в автоматическом режиме при помощи кнопки «Подобрать документы РК» или в ручном режиме. Рекомендуем самостоятельно контролировать корректность подбора документа. При загрузке таких документов в начальном налоговом документе будут заполнены номер и дата входящего документа. 
Замечание. Красным цветом выделяются документы,  для которых нарушен предельный срок регистрации в ЕРНН.
