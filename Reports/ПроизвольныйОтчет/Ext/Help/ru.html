Справочник «Произвольные отчеты»

Справочник предназначен для создания новых отчетов. Создание и изменение общей настройки отчетов доступно пользователям с ролью "Администрирование дополнительных форм и обработок". Также таким пользователям доступен обмен отчетами: выгрузка и загрузка отчетов.

Общая настройка отчета

Первым этапом создания отчета является создание схемы компоновки данных.

Для наборов данных "Объект" можно настроить источники данных. Если в списке источников данных присутствует несуществующий набор данных, то он выделяется особым цветом. Такой источник данных необходимо либо удалить, либо указать другой набор данных.

После создания отчета изменить общие настройки может только администратор отчета.

Для отчета можно использовать "Конструктор настройки показателя монитора эффективности".
