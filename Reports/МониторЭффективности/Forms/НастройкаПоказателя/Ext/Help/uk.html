Настройка показателя

Форма предназначена для настройки показателя.

В форме необходимо заполнить поля "Наименование", "Отчет". При необходимости выбрать настройку (вариант отчета).

В поле "Отчет" выбирается произвольный отчет, который представляет собой показатель монитора эффективности. Показатели можно создавать самостоятельно при помощи конструктора или загрузить из демо-базы.

 

Настройка параметров

Среди параметров обязательно присутствуют "Смещение периода" и "Периодичность анализа". Если их не заполнять, то будут использоваться значения по умолчанию.

Остальные параметры заполняются индивидуально.

 

Настройка индикаторов

При настройке индикаторов необходимо ввести "Границы тренда" и "Границы состояния". При необходимости можно изменить индикатор каждого интервала.

 

Расшифровки

Настройка расшифровки предназначена для ввода дополнительных расшифровок показателя.

По умолчанию показатель расшифровывается отчетом, который выбран в поле "Показатель".
