Регистр сведений "Значения показателей схем мотивации"

В данном регистре хранятся значения показателей схем мотивации. По данным этого регистра заполняются значения показателей схем мотивации в документах: "Начисление зарплаты работникам" и "Начисление зарплаты работникам организаций".

Способы заполнения регистра сведений "Значения показателей схем мотивации"
  • Загрузка данных из файлов форматов: mxl, xls, dbf, txt. Для того чтобы воспользоваться этой возможностью, нажмите на кнопку "Заполнить" панели инструментов, выберете пункт меню "Заполнить по данным внешнего файла (mxl, xls, dbf, txt)"
  • Полуавтоматическое заполнение по плановым начислениям. Для того чтобы воспользоваться этой возможностью, нажмите на кнопку "Заполнить" панели инструментов и выберете пункт меню "Заполнить по плановым начислениям", нажмите на кнопку    в открывшемся окне, выбрать загружаймый файл, нажать на кнопку "Загрузить".
  • Ручной ввод данных.
