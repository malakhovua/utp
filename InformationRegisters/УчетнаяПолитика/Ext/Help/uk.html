Регистр сведений «Учетная политика»

Регистр предназначен для хранения информации об учетной политике предприятия «в целом». Сведения используются главным образом в управленческом учете, но часть из них влияет на ведение бухгалтерского и налогового учета.

Содержит следующие показатели:

Ведение учета по проектам - установка этого флажка означает, что в системе будет возможно использовать дополнительную детализацию продаж, закупок, движения денежных средств в  разрезе проектов.

Стратегия списания партий товаров по хронологии - определяет стратегию выбора партий при списании в управленческом учете: ФИФО, ЛИФО, по средней.

Стратегия списания партий товаров по статусам партии - параметр настройки, влияющий на метод списания партий ценностей одинаковой номенклатуры (характеристик, серий) в условиях, когда имеются собственные товары и товары, принятые на комиссию: можно отдавать предпочтение при списании либо собственным товарам, либо товарам, принятым на комиссию.

Вести партионный учет по складам - если выбрано значение «да» (установлен флажок), то партионный учет ведется в разрезе каждого склада отдельно, и, соответственно, методика списания запасов (ФИФО, ЛИФО или по средней стоимости) применяется по отношению к каждом складу (иначе - ко всему предприятию или организации в целом). Установленное значение влияет на все виды учета (управленческий, бухгалтерский, налоговый, международный).

Списывать партии при проведении документов - если выбрано значение «да» (установлен флажок), то списание партий запасов производится при проведении документов по движению запасов, если «нет» (флаг не установлен), то списание партий осуществляется регламентно с помощью обработки «Проведение по партиям». Установленное значение влияет на все виды учета (управленческий, бухгалтерский, налоговый, международный).

Параметр ABC классификации покупателей - параметр, по которому будут классифицироваться покупатели по важности при АВС-Классификации.

Параметр распределения покупателей по стадиям взаимоотношений - параметр, по которому будут классифицироваться постоянные покупатели при классификации покупателей по стадиям взаимоотношений.

Способ расчета себестоимости производства. При выборе способа "По переделам", расчет себестоимости полуфабрикатов и продукции производится в том порядке, в котором они выпущены и списаны в дальнейшее производство: полуфабрикаты 1-го передела, 2-го передела и т.д. до готовой продукции; порядок переделов определятся автоматически. При выборе способа "По подразделениям", последовательно проводится расчет себестоимости продукции каждого подразделения; порядок закрытия подразделений определяется в документе "Установка порядка подразделений для закрытия счетов".

Распределение затрат основного производства для услуг сторонним заказчикам. Определяет, какой измеритель обычно используется при учете производственных услуг, реализуемых на сторону:

  • При выборе варианта "По плановой себестоимости выпуска" оказание услуг отражается "Актом об оказании производственных услуг", услуги в течение месяца учитываются по плановой себестоимости; при расчете фактической себестоимости производственные затраты подразделения по номенклатурной группе распределяются между видами продукции пропорционально плановой себестоимости.
  • При выборе варианта "По выручке" оказание услуг отражается "Реализацией товаров и услуг", услуги в течение месяца учитываются по цене реализации; при расчете фактической себестоимости производственные затраты распределяются между видами продукции пропорционально выручке.
  • При выборе варианта "По плановой себестоимости и выручке", различные виды услуг учитываются в различных измерителях. Услуги, учитываемые по плановой себестоимости и по выручке, регомендуется относить к различным номенклатурным группам.
