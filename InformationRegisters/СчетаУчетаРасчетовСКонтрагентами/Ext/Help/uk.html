Регистр сведений "Счета учета расчетов с контрагентами"

Для того чтобы в документах каждый раз не выбирать вручную счета расчетов с контрагентами эти счета можно определить один раз. После этого такие счета будут по умолчанию использоваться при подстановке в документы в качестве счетов расчетов с контрагентами.

Счет по умолчанию указывается в привязке к организации, контрагенту, договору. Причем, можно указать приоритеты выбора правила подстановки. Например, если в документе установлены значения организации, контрагента, договора, то в списке правил, определяющих выбор счета по умолчанию в первую очередь будут анализироваться правила, в которых значение организации, контрагента, договора совпадают со значениями, установленными в документе. При создании нового документа, когда не указана ни организация, ни контрагент, ни договор из списка правил будет выбрано то, в котором значения параметров организация, контрагент, договор отсутствуют.

Согласно приоритетам действия правил при создании нового документа, когда неизвестны данные об организации, контрагенте, договоре, будет действовать правило №4; при указании организации счет выберется согласно правилу №3; после выбора контрагента в документе сработает правило №2; после указания договора – правило№1.
