Справочник «Способы отражения зарплаты в регламентированном учете»

Здесь задаются хозяйственные операции, которые можно поставить в соответствие расчетным величинам.

 

Хозяйственная операция состоит из одной проводки по бухгалтерскому регистру (или пары транзитных проводок) и статей затрат для налогового учета.

 

Особое внимание следует обратить на механизм формирования взносов в фонды на основании проводок начислений. Общая структура проводки по фондам остается такая же как и у базового начисления, изменяется только статья затрат на значение реквизита "Статья затрат для взносов" (для бухгалтерского учета) и стать налоговой декларации по налогу на прибыль на значение реквизитов "Статья декларации по ННП для взносов" (для налогового учета).

 

Если в реквизите "Налоговое назначение " для элемента установлен вид налоговой деятельности "Не  хозяйственная (не облагаемая)", то движений в налогом учете сформировано не будет.

 

Порядок отражения взносов в фонды социального страхования на ФОТ:

Для правильного формирования проводок по взносам в Фонды социального страхования необходимо выполнить следующие первоначальные настройки:

 

- заполнить справочник "Способы отражения зарплаты в регламентированном учете". Должны быть созданы проводки для всех видов начислений, способы отражений для взносов в фонды уже присутствуют в справочнике как предопределенные элементы; 

 

- заполнить справочник "Статьи затрат" - необходимо помнить о том, что когда бухгалтер создает Статью затрат с "Видом затрат" = "Оплата труда", необходимо создать еще и затрату с "Видом затрат" = "Отчисления на социальные нужды". Если интересна подробная информация о формировании зартрат, то  можно создать отдельную затрату для каждого вида взноса в социальные фонды;

 

-  заполнить регистр сведений "Отражение Взносов на ФОТ в регл. учете".. Здесь каждому "Налогу" (Взнос в фонд) ставится в соответствие"Статья затрат" (с  "Видом затрат" = "Отчисления на социальные нужды" );

 

Замечание: Эти операции выполняются при начальной настройке программы.

 

- С помощью документов "Прием на работу в организацию"  и "Кадровое перемещение организаций" необходимо задать характер проводок для начислений.

 

После того как бухгалтер добился того,  что в документе "Отражение зарплаты в регл учете" на странице "Начисления" правильно формируются все проводки по начислениям (заполнены все реквизиты, типа "Счет..." и "Субконто"), можно приступить к анализу заполнения табличной части "Взносы ФОТ".

 

Для целей формирования проводки по взносам в фонды реквизиты проводки анализируются в таком порядке:

 

- если установлен флаг "Получать взносы основного начисления" в строке документ (табличные части "Начисления"или "Дополнительные начисления" документа "Отражение зарплаты в регл учете" ) то для анализа проводок используется не значение строки документа, а значение элемента справочника "Способы отражения в регламентированном учете", которое поставлено в соответствие работнику в таблице "Учет основного заработка работников организаций в регламентированном учете" (изменяется документами "Прием на работу в организацию"  и "Кадровое перемещение организаций" );

Замечание: Сейчас этот реквизит должен быть заполнен только для отражения в регламентированном учете "Начислений по больничным листам за счет соцстраха".

- если у проводки "Счет Дт (взносы)" - не определен, тогда используются значения реквизитов "Счет Дт" и соответствующие ему субконто;

 

- для заполнения реквизита "Статья затрат" анализируется субконто проводки начисления типа "Статья затрат". По Виду взносов в фонды получаем : "Характер затрат". По этой паре определяется "Статья затрат" для взноса на ФОТ (регистр сведений  "Отражение Взносов на ФОТ в регл. учете").

 

Замечание:

Субконто типа "Работники организации" и  "Подразделения" можно не заполнять, субконто других видов должны быть заполнены обязательно.
