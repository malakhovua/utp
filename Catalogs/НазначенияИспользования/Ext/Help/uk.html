Справочник "Назначения использования"

Справочник предназначен для указания назначений использования переданных в эксплуатацию малоценных активов.

Заполнение реквизитов

  • Наименование - произвольный текст. Наименование назначения использования.
  • Вид малоценки - к какой категории  будет относится актив по назначению использования:
    • Малоценный быстроизнашивающийся предмет;
    • Малоценный необоротный актив;
    • Библиотечный фонд.
  • Количество по нормативу выдачи - количество по нормативу. Значение реквизита используется для подстановки по умолчанию количества в документы.
  • Единица - единица измерения количества (базовая единица измерения номенклатуры, владельца данного назначения)
  • Срок полезного использования - срок полезного использования малоценного актива.
  • Способ отражения расходов - способ отражения в затратах суммы погашенной стоимости малоценного актива.
  • Счет передачи - счет бухгалтерского учета по назначению использования (счет на котором будет числится актив после передачи в эксплуатацию). Для МБП - "222", поскольку именно с этого счета оно спишется на затраты в суммовом эквиваленте, а на этот счет спишется количество передаваемого материала. Для малоценных необоротных активов и библиотечных фондов были заведены специальные субсчета с аналитикой по номенклатуре и количественным учетом ("1122"," 1112"). Эти счета подставляются по умолчанию, при изменении вида материала.
  • Счет амортизации - счет бухгалтерского учета амортизации по назначению использования. На этом счете будет числится амортизация, начисленная после передачи в эксплуатацию малоценных нематериальных активов и библиотечных фондов.
