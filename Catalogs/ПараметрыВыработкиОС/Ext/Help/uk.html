Справочник "Параметры выработки ОС"

Справочник содержит перечень параметров, по которым в системе учитывается выработка оборудования. Выработка по таким параметрам используется для расчета амортизации, если для основного средства выбран производственный способ расчета (пропорционально работам). Учет фактической выработки отражается документом "Выработка ОС".
