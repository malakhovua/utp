Настройка параметров учета

К параметрам учета (константам) относятся настройки информационной базы, которые действуют для всех организаций предприятия и без крайней необходимости не изменяются после начального заполнения базы.

Закладка "Общие"

На закладке расположены флаги, определяющие использование определенных подсистем конфигурации. При снятии флага отключается возможность работы с соответствующей подсистемой.

Интервал проверки напоминаний в секундах - интервал, с которым будет проверяться системой наличие записей для напоминания пользователям.

Рабочий день пользователей - время начала и окончания рабочего дня.

Флаги "Использовать характеристики", "Использовать серии"  устанавливаются в том случае, если в предприятии ведется учет товаров по сериям и характеристикам. Если эти флаги не установлены, то в документах и справочниках не будут показываться соответствующие поля для учета серий и характеристик.

Также на предприятии можно отключить учет по внутренним заказам.

В конфигурации предусмотрена возможность оформления документов поступления и реализации на несколько складов с указанием склада в табличной части документа. Эта возможность будет доступна только в том случае, если установлена соответствующая настройка в поле "Указание складов в табличной части документа".

Настройка шаблонов быстрого ввода информации о контрагенте - пользователь может заранее задать поля быстрой регистрации нового контрагента из документа "Событие".

Закладка "Печать, единицы"

Дополнительная колонка печатных форм документов - информация в данном поле определяет ту дополнительную информацию, которая выводится в печатных формах документов и в отчетах.

  • Код - в печатных формах документов и в отчетах будет дополнительно выводиться код, заданный для позиции номенклатуры. 
  • Артикул - в печатных формах документов и в отчетах будет дополнительно выводиться артикул, заданный для позиции номенклатуры. 
  • Не выводить - дополнительной информации выводиться не будет.

Единица измерения веса -  в данном поле необходимо задать краткое наименование единицы веса, используемой для измерения веса позиции номенклатуры в конфигурации.

Единица измерения объема - в данном поле необходимо задать краткое наименование единицы объема, используемой для измерения объема позиции номенклатуры в конфигурации.

Закладка "Валюты"

Валюта регламентированного учета — базовая валюта, курс которой всегда равен 1 (для Украины —

гривни). Курсы всех других валют выражаются в единицах валюты бухгалтерского учета.

Валюта управленческого учета — валюта, используемая для оценки данных в управленческом учете. Обычно в качестве валюты управленческого учета указывают доллар США.

Закладка "Электронная почта"

Пользователь может указать вариант использования электронной почты. Можно пользоваться либо встроенным в конфигурацию почтовым клиентом, либо основным почтовым клиентом операционной системы (Microsoft Outlook, Outlook Express, The BAT! и пр.). Для встроенного почтового клиента задается администратор учетных записей, который может редактировать учетные записи других пользователей.

Закладка "Шаблоны телефонов"

Для единообразия вводимых телефонных номеров всеми пользователями в контакную информацию, можно задать единые шаблоны. При изменении или добавлении телефонных номеров, они будут приводиться к указанным шаблонам.

Закладка "Штрих-коды"

На этой закладке задаются параметры для автоматического формирования штрих-кодов на товары.
В конфигурации предусмотрено автоматическое шрихкодирование товаров для типа штрих-кода EAN-13.

В том случае, если на торговом предприятии ведется учет по штрих-кодам весового товара, то необходимо задать префикс штрих-кода весового товара. По этому параметру при заполнении информации о весовом товаре с помощью сканера штрих-кодов в документе будет определяться, что товар является весовым, и в документе будет заполняться вес товара, заданный в штрих-коде. 

Заполнение префикса для штучного товара позволит формировать этикетки на товары с внутренним штрих-кодом, отличным от штрих-кода поставщика.

Для каждой единицы измерения товаров задается свой штрих-код. Если на торговом предприятии допускается назначать разным товарам или товарам с разными единицами измерения один и тот же штрих-код, то необходимо установить флаг «Разрешить назначение неуникальных штрих-кодов».

Закладка "Параметры бухгалтерского учета"

На этой закладке расположена панель с настройками параметров бухгалтерского учета.

Закладка "План счетов"

На этой закладке устанавливается признак использования счетов-классов в качестве групп.

Примечание

При установке флага "Использовать классы счетов в качестве групп" в плане счетов бухгалтерского учета счета-классы (1 "Необоротные активы", 2 "Запасы" и т.д.) будут использоваться как группы счетов. То есть по  таким счетам можно будет получить информацию об остатках и оборотах в отчетах.

При снятии указанного флага счета имеющие код, состоящий из двух цифр (10 "Основные средства",  20 "Производственные запасы" и т.д.), станут счетами первого уровня и не будут подчинены счетам-классам.

Закладка "Аналитический учет МПЗ"

На этой закладке устанавливаются следующие параметры:

  • признаки ведения партионного учета (флаг "Вести партионный учет"), складского учета (флаг "Вести складской учет") и суммового учета по складам (флаг "Вести суммовой учет по складам") для бухгалтерского учета;
  • признаки ведения складского учета (флаг "Вести складской учет") для налогового учета;
  • для организаций, оценивающих товары в розничной торговле по продажным ценам (соответствующий признак устанавливается в учетной политике бухгалтерского учета) — признак использования оборотной аналитики по номенклатуре (флаг "Использовать оборотную аналитику по номенклатуре") и признак учета товаров в разрезе ставок НДС (флаг "Учитывать товары в разрезе ставок НДС");
  • признак необходимости учета возвратной тары (флаг "Использовать возвратную тару").

Примечание

При установке флага "Вести партионный учет" в бухгалтерском плане счетов на счетах, предусматривающих аналитический учет по субконто "Номенклатура" (20 "Материалы", 28 "Товары", 26 "Готовая продукция" и т.д.), автоматически будет установлено субконто "Партии". При снятии указанного флага субконто "Партии" будет автоматически убрано с этих счетов. При этом вся информация о партиях в проводках будет потеряна.

При установке флага "Вести складской учет" в бухгалтерском плане счетов на счетах, предусматривающих аналитический учет по субконто "Номенклатура" (20 "Материалы", 28 "Товары", 26 "Готовая продукция" и т.д.), автоматически будет установлено субконто "Склады". При снятии указанного флага субконто "Склады" будет автоматически убрано с этих счетов. При этом вся информация о складах в проводках будет потеряна.

На закладке "Аналитический учет МПЗ" можно отключить автоматический контроль отрицательных остатков товаров и других запасов (флаг "Отключить контроль отрицательных остатков МПЗ"). 

Закладка "Прочий аналитический учет"

Аналитический учет расчетов с контрагентами

В этой группе устанавливается признак детализации расчетов с контрагентами до документов (флаг "Вести расчеты по документам").

Примечание

При установке флага "Вести расчеты по документам" в плане счетов на счетах расчетов с контрагентами (63 "Расчеты с поставщиками и подрядчиками", 36 "Расчеты с покупателями и заказчиками", субсчета по расчетам счетов 377 "Расчеты с другими дебиторами", 681 "Расчеты с другими кредиторами" и т.д. ) автоматически будет установлено дополнительное субконто "Документы расчетов с контрагентами". При снятии указанного флага субконто "Документы расчетов с контрагентами" будет автоматически убрано с этих счетов. Вся информация о документах расчетов с контрагентами в проводках при этом будет потеряна.

Учет по обособленным подразделениям

В этой группе устанавливается признак детализации учета наличных денежных средств по обособленным подразделениям  (флаг "Вести аналитический учет денежных средств по обособленным подразделениям").

Примечание

При установке флага "Вести аналитический учет денежных средств по обособленным подразделениям" на счетах учета денежных средств в кассе будет использоваться субконто "Обособленные подразделения".

Закладка "Производство и затраты"

Тип цен плановой себестоимости номенклатуры. Указывается тип цен, цены которого используются при опредлении плановой себестоимости:

  • Продукции (полуфабрикатов, услуг) - плановая себестоимость указывается в документах "Отчет производства за смену", "Акт об оказании производственных услуг". При расчете фактической себестоимости при закрытии месяца плановая себестоимость используется как база распределения производственныхзатрат;
  • Товаров, закзанных покупателем, и товаров, заказанных у поставщиков - плановая себестоимость указывается в заказах и счетах. Контролируется отклонение цены продаи (покупки) товаров от их плановой себестоимости

Распределение затрат основного и всомогательного производства для услуг собственным подраздлениям. Определяет, какой измеритель обычно используется при учете внутренних услуг в документе "Отчет производства за смену": плановая себестоимость или количество. При расчете фактической себестоимости при закрытии месяца производственные затраты будут распределятья между видами услуг пропорционально их плановой себестоимости или количеству.

Комиссионное вознаграждение. Статья затрат, которая по умолчанию используется в документах "Отчет комиссионера о продажах" при отнесении на затраты комиссионного вознаграждения

Покупка-продажа валюты. Статья затрат, которая по умолчанию используется в документах "Покупка-продажа валюты" в управленческом учете.
