Обработка "Выгрузка данных в идентичную конфигурацию"

Выгрузка данных в идентичную конфигурацию "Управление торговым предприятием для Украины" производится с помощью одноименного пункта в меню "Сервис".

При выгрузке данных необходимо указать период, за который необходимо выгружать документы, справочники и регистры.

Выгрузка данных производится в файл в формате xml, имя которого указывается в настройках выгрузки.

При осуществлении выгрузки на закладке "Отбор" можно произвести отбор данных по складу, подразделению или организации.

На закладке " Правила выгрузки данных" можно отметить те документы , справочники и регистры сведений, которые необходимо выгружать. В том случае, если у какого -то объекта флаг не установлен, то тогда данный объект будет выгружаться только в том случае, если он используется в выгружаемом документе или справочнике в качестве ссылки.

Загрузка данных в идентичную конфигурацию производится с помощью пункта меню "Универсальный обмен данными в формате XML". Загрузка данных производится на закладке "Загрузка данных". Необходимо указать тот файл, в который выгружались данные из идентичной конфигурации.

 

ВНИМАНИЕ! Для корректной загрузки данных в базе-приемнике необходимо установить настройки параметров учета аналогичные настройкам базы-источника (до начала загрузки данных).
