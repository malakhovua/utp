Обработка "Формирование налоговых документов по рознице"

Обработка "Формирование налоговых документов по рознице" предназначена для заполнения налоговых накладных по розничным продажам.

Для заполнения используюется информаци о принятой розничной выручки за выбранный день (документы "Приходный кассовый ордер" с видом операции "Приём розничной выручки").

Документы формируются на сумму остатка розничной выручки (по указанную дату), на который не были выписаны налоговые документы. 

Налоговые накладные будут выписаны от имени обособленного подразделения, указанного на форме обработки.
