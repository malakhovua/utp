Документ «Начисление отпуска работникам организаций»

Документ «Начисление отпуска работникам организации»  регистрирует замену плановых начислений определенного работника на определенный период начислением оплаты по среднему заработку со способом расчета «По среднему заработку для отпуска». Документ также можно использовать для расчета компенсации отпуска.

Сведения, необходимые для расчета начислений, указываются в шапке документа.  Порядок расчета и внешний вид документа определяет реквизит "Вид документа":

  • Отпускные - в шапке документа появляется список, в который можно заполнить виды расчета и периоды отпуска (периоды должны следовать один за другим без прерываний, если между отпусками есть рабочие дни, то стоит заполнить 2 отдельных документа);
  • Компенсация отпуска -  оформляется в случае, когда работнику компенсируется только сумма отпуска, но сотрудник продолжает работать;
  • Отзыв из отпуска - при заполнении этого вида документа, необходимо в поле "Первичный отпуск" заполнить документ отпуск из которого сотрудника отзывают, а также список видов расчетов и периодов, которые должны быть отсторнированы;
  • Возврат отпускных при увольнении - этот вид документа используется если при увольнении выяснилось, что сотрудник отгулял отпускных больше, чем ему положено на дату увольнения. Если же оказалось, что сотруднику положены дополнительные дни отпуска, то лучше воспользоваться видом "Компенсация отпуска".

 

Страница "Расчет среднего заработка"

Страница содержит табличную часть и дополнительные параметры, которые определяют порядок заполнения и расчета табличной части.

Важно! Если в регистре сведений "Параметры расчета заработной платы организации" для организации включен флаг "Есть годовая премия", то в группе "Дополнительные параметры расчета средней" станут видимыми и доступными реквизиты "Расчет премии за год", а при заполнении табличной части в нее будут добавлены записи с видом расчета "По годовой премии (для отпуска и средней)".
Если в том же регистре сведений включен флаг "Есть премия", в табличную часть будут также добавлены виды расчета "По премиям (учитывают пропорционально)" и "По премиям (учитывают полностью)".

По результатам заполнения и расчета табличной части "Расчет среднего" рассчитывается "Сумма средней". Порядок расчета зависит от значения реквизита "Вид учета времени для средней":

  • По рабочим дням;
  • По рабочим часам;
  • По календарным дням - значение по умолчанию.

     

    Страница "Начисления"

    Меню "Заполнить" и "Рассчитать" позволяют перезаполнить и рассчитать начисления, учитывая ручные изменения в расчете средней.

    Для видов расчета отпускных в табличной части "Начисления" будут автоматически заполнены параметры "Средняя" и "Вид учета времени для средней", они заполняются данными со страницы "Расчет среднего заработка".

    Важно! В случае, когда сотрудник принят на внутреннее совместительство в документе для вида "Отпускные" и "Отзыв из отпуска" в таблице Начисления должны возникнуть 2 записи по каждому виду расчета но с различными назначениями (основному и по всем внутренним совмещениям). Это необходимо для обеспечения работы механизма вытеснений. Суммы будут рассчитаны только в записях по основному назначению, остальные записи должны присутствовать в документе формально.
