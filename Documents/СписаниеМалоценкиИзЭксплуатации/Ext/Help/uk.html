Документ "Списание малоценки из эксплуатации"

Документ предназначен для отражения в учете операций списания из эксплуатации малоценки.

При вводе документа в шапке необходимо указать следующие реквизиты:

  • Подразделение. Производственное подразделение организации, из которого списывается из эксплуатации малоценка. Значение реквизита выбирается из справочника "Подразделения". Реквизит является обязательным для заполнения.

В табличной части необходимо указать список списываемых активов и количество. При этом заполняются следующие реквизиты:

  • Малоценный актив. Списываемая из малоценка. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Назначение использования. Назначение использования малоценки. Значение реквизита выбирается из справочника "Назначения использования" и является обязательным для заполнения.

  • Физическое лицо . Физическое лицо, у которого списывается малоценка. Значение реквизита выбирается из справочника "Физические лица" и является обязательным для заполнения.

  • Количество. Количество списываемой малоценки.

В документе организовано автоматическое заполнение по остаткам. При нажатии на кнопку заполнения командной панели открывается форма отбора остатков, в которой можно задать необходимые условия для заполнения табличной части документа.  При установленном флажке "Указывать партию" заполнение производится в разрезе партий малоценки в эксплуатации.

Если списание производится ранее истечения срока полезного использования, то для корректировки НДС можно ввести на основании данного документа налоговую накладную.

Особенности проведения документа

В бухгалтерском учете формируется проводка по списанию со счета МЦ и по закрытию амортизации если вид малоценки не МБП.
