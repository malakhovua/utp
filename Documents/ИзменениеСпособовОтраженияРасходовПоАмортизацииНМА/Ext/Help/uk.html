Документ "Изменение способов отражения расходов по амортизации НМА"

Документ позволяет изменить способ отражения расходов по амортизации по Нематериальным активам.

Данный документ не формирует никаких проводок, он лишь изменяет значение соответствующих ресурсов регистра сведений, хранящего информацию об НМА "Способы отражения расходов по амортизации (бухгалтерский учет)" ).

Влияет на распределение начисленной амортизации бухгалтерскому учету.
