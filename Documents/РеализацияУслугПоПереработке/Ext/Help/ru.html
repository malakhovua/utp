Документ "Реализация услуг по переработке"

Документ предназначен для отражения в бухгалтерском и налоговом учете операции оказания услуг по выпуску продукции из давальческого сырья и отражения отчета об использованных при выпуске продукции давальческих материалах.

Общие реквизиты документа

  • Контрагент.  Заказчик переработки сырья. Значение реквизита выбирается из справочника "Контрагенты". Реквизит является обязательным для заполнения.

  • Договор.  Договор с заказчиком переработки сырья с видом "С покупателем".

  • Закладка "Продукция (услуги по переработке)"

    На закладке указывается продукция из давальческого сырья, услуги, по выпуску которой, оформляются данным документом.

    • Номенклатура. Выпущенная продукция из давальческого сырья по стоимости услуг по ее производству. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

    • Спецификация. Спецификация продукции, если для данной продукции заведена спецификация. Значение реквизита выбирается из справочника "Спецификации номенклатуры". Такая информация позволит автоматически рассчитать состав материалов, необходимый для производства указанной продукции заполнить закладку "Материалы заказчика".

    • Содержание услуги. Дополнительная информация об оказанной услуге.  

    • Количество. Количество выпущенной продукции.
    • Цена. Цена реализации услуги по выпуску продукции.
    • Сумма. Сумма реализации услуги по выпуску продукции.
    • Стоимость (плановая). Плановая цена (в валюте регламентированного учета) продукции, произведенной из сырья заказчика.
    • Сумма (плановая). Плановая стоимость (в валюте регламентированного учета) продукции, произведенной из сырья заказчика.
    • Счет учета (БУ). Счет бухгалтерского учета производства продукции из давальческого сырья (например, 234). Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Счета учета номенклатуры".
    • Схема реализации. Схема реализации услуги. Значение заполняется автоматически при выборе продукции на основании данных регистра сведений "Счета учета номенклатуры".

    Предусмотрено заполнение данной табличной части по кнопке "Заполнить". Заполнение производится по отчету производства за смену, который будет указан из предложенного списка и в котором счет учета готовой продукции будет 234 или по остаткам счета 234.

    Закладка "Материалы заказчика"

    На закладке указываются материалы, принятые в переработку, которые были использованы при выпуске продукции.

    • Номенклатура. Материал, принятый в переработку. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

    • Количество. Количество материала.
    • Счет учета (БУ). Счет бухгалтерского учета материала, принятого в переработку и переданного в производство (например, 0222). Значение реквизита выбирается из плана счетов бухгалтерского учета.

    Предусмотрено заполнение данной табличной части по кнопке "Заполнить". Заполнение производится давальческими материалами, которые были переданы в производство и учтены на счете 0222 "Материалы, переданные в производство" или состав материалов можно рассчитать по спецификациям и продукции, указанным на закладке "Продукция и услуги".

    Закладка "Счета учета расчетов"

    На закладке в документе заполняются «Счет учета расчетов с контрагентом» и «Счет учета расчетов по авансам». Эти счета заполняются в соответствии с заданными в регистре сведений «Контрагенты» значениями по умолчанию для каждого контрагента. При необходимости пользователь может изменить корреспонденцию счетов.

    Особенности проведения документа

    По данным табличной части "Продукция и услуги" отражается реализация услуг по переработке по плановым ценам и списание готовой продукции, произведенной из материалов заказчика со счета 234 по плановым ценам.

    По данным табличной части "Материалы заказчика" отражается списание материалов заказчика со счета 0222, как использованных на производство продукции для заказчика, отчет по которым принят заказчиком.
