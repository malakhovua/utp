Документ "Начисление коммунального налога"

Документ «Начисление коммунального налога» предназначен для автоматизированного начисления коммунального налога и формирования проводок по бухгалтерскому и налоговому учету.

Из документа можно воспользоваться обработкой "Формирование платежных документов по взносам в фонды" (Перейти -> Формирование платежных документов по взносам в фонды).
