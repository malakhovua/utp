Документ «Начисление по больничному листу»

Документ «Начисление по больничному листу» регистрирует замену плановых начислений начислением оплаты по среднему заработку указанному в документе работнику на указанный период.

В шапке документа указываются необходимые реквизиты, в частности, Организация, Месяц начисления. Необходимо указать Сотрудника а также период "На больничном с" и "по", в случае необходимости изменить причину (по умолчанию "Общее заболевание"). Если больничный продлевается, то в поле "Первичный б/л" следует заполнить первоначальный документ, из него будут получены все данные о расчете средней.

Полный расчет изаполнение документа можно выполнить кнопкой "Заполнить и рассчитать все".

Страница "Расчет среднего заработка"

На странице "Расчет среднего заработка" можно проверить и пересчитать среднюю.

В реквизите "Сумма средней" отражается результат расчета средней.  Вид времени за который отображена средняя определяется по значению реквизита "Вид учета времени для средней":

  • По рабочим дням - значение для больничного по умолчанию;
  • По рабочим часам - если у сотрудника график с суммированным учетом времени, или если его основное начисление - оплата по часовому тарифу;
  • По календарным дням.

     

    При изменении периода больничного будет задан вопрос о необходимости  перерасчета "Дополнительных параметров расчета среднего":

    • "Период средней" - как правило это 6 месяцев, но в некоторых случаях может быть меньше (если сотрудник не отработал 6 месяцев);
    • "Вид учета времени для средней";
    • "Сумма предела";
    • "Число месяцев" - скорее информационный;
    • "Процент оплаты" - зависит от причины больничного:  для причины "По беременности и родам" всегда 100%, для остальных причин процент оплаты зависит от длительности непрерывного трудового стажа сотрудника. Сведения о стаже отображаются в информационной строке под реквизитом "Процент оплаты". Изменить сведения о стаже можно в списке "Начальный стаж сотрудников организации" (В меню документа "Перейти"-> "Стаж сотрудника").

    Данные по расчету среднего можно рассчитать поэтапно на странице "Расчет среднего заработка". Кнопки "Заполнить" и "Рассчитать" табличной части расчета среднего заполняют табличную часть помесячно, по две записи на месяц:

    • (больничный) По заработку для расчета больничных;
    • (больничные пропорционально) По начислениям, которые учитываются пропорционально отработанному времени.

    Если сотрудник заболел в месяце приема на работу, средний заработок будет определен исходя из оклада: размер оклада на момент начала заболевания будет разделен на среднемесячное количество рабочих дней.

    Табличную часть можно заполнить вручную, после чего следует рассчитать сумму средней с помощью кнопки "Рассчитать".
    Если данных для расчета среднего заработка в базе недостаточно, сумму средней можно также заполнить вручную.

    Страница "Начисления"

    Для видов расчета "Оплата по среднему БЛ (за счет организации)", "Оплата по среднему БЛ" в табличной части "Начисления" будут автоматически заполнены параметры "Средняя", "%" и "Вид учета времени для средней", они заполняются данными со страницы "Расчет среднего заработка".

  • Важно! В случае, когда сотрудник принят на внутреннее совместительство в таблице Начисления должны возникнуть 2 записи по каждому виду расчета, но с различными назначениями (основному и по всем внутренним совмещениям). Это необходимо для обеспечения работы механизма вытеснений. Суммы будут рассчитаны только в записях по основному назначению, остальные записи должны присутствовать в документе формально.

    Флаг "Применять ограничение размером мин. зарплаты" необходимо возводить в тех случаях, когда бухгалтеру известно, что сотрудник последние 6 месяц не платил страховые взносы и средняя сотрудника должна быть рассчитана исходя из минимальной зарплаты.
