Документ "Получение из переработки"

Документ предназначен для отражения в бухгалтерском и налоговом учете операции получения услуг по выпуску продукции сторонним переработчиком, самой продукции и отражения отчета об использованных материалах.

Общие реквизиты документа

  • Склад. Место хранения, на который поступает готовая продукция. Значение реквизита выбирается из справочника "Склады". Реквизит является обязательным для заполнения.

  • Контрагент.  Переработчик сырья. Значение реквизита выбирается из справочника "Контрагенты". Реквизит является обязательным для заполнения.

  • Договор.  Договор переработчика с видом "С поставщиком".

  • Подразделение на котором будет отражено производство продукции силами стороннего переработчика.

  • Номенклатурная группа, к которой отнесена продукция, произведенная переработчиком.

  • Счет затрат. Счета основного или вспомогательного производства, на котором будет отражено производство продукции силами стороннего переработчика.  

Закладка "Продукция"

На закладке указывается перечень  продукции, произведенной переработчиком.

  • Номенклатура. Продукция, выпущенная сторонним переработчиком. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

  • Спецификация. Спецификация продукции, если для данной продукции заведена спецификация. Значение реквизита выбирается из справочника "Спецификации номенклатуры". Такая информация позволит автоматически рассчитать состав материалов, необходимый для производства указанной продукции заполнить закладку "Материалы".
  • Количество. Количество продукции.
  • Цена (плановая). Плановая цена (в валюте регламентированного учета) производства продукции сторонним переработчиком.
  • Сумма (плановая). Плановая стоимость (в валюте регламентированного учета) услуги по выпуску продукции сторонним переработчиком.
  • Счет учета (БУ). Счет бухгалтерского учета отданных в переработку материалов. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Счета учета номенклатуры организаций".

     

    Закладка "Услуги"

    На закладке указывается перечень  услуг, оказанных переработчиком по преработке сырья в готовую продукцию.

    • Номенклатура. Оказанная услуга. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

    • Содержание услуги. Дополнительная информация об оказанной услуге. 

    • Статья затрат. Статья затрат, по которой услуги по переработке будут отнесены на затраты производства. Значение реквизита выбирается из справочника "Статьи затрат".

    • Количество. Количество оказанных услуг.
    • Цена. Цена оказанной услуги.
    • Сумма. Стоимость оказанной услуги.

    Закладка "Материалы"

    На закладке указываются материалы, которые были использованы переработчиком при выпуске продукции.

    • Номенклатура. Материал, переданный в переработку. Значение реквизита выбирается из справочника "Номенклатура" и является обязательным для заполнения.

    • Количество. Количество материала.

    • Цена. Залоговая цена материла, по которому он был передан в переработку.

    • Сумма. Залоговая стоимость материала.

    • Счет учета (БУ). Счет бухгалтерского учета отданных в переработку материалов. Значение реквизита выбирается из плана счетов бухгалтерского учета и заполняется автоматически при выборе продукции на основании данных регистра сведений "Счета учета номенклатуры организаций".

    Предусмотрено заполнение данной табличной части по кнопке "Заполнить". Заполнение производится материалами, которые согласно спецификации необходимы для производства продукции, указанной на закладке "Продукция".

    Закладка "Счета учета расчетов"

    На закладке в документе заполняются «Счет учета расчетов с контрагентом» и «Счет учета расчетов по авансам». Эти счета заполняются в соответствии с заданными в регистре сведений «Контрагенты» значениями по умолчанию для каждого контрагента. При необходимости пользователь может изменить корреспонденцию счетов.

    Особенности проведения документа

    По данным табличной части "Продукция" производится поступление из производства на склад готовой продукции в плановых ценах.

    По данным табличной части "Услуги" производится принятие услуг переработки сырья сторонней организацией на расходы основного или вспомогательного производства.

    По данным табличной части "Материалы" производится списание на затраты основного или вспомогательного производства материалов, ранее переданных в переработку на сторону и учтенных на счете 206 "Материалы, переданные в переработку на сторону".
